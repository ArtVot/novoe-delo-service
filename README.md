# 🛠 Проект: Информационный ресурс сервисного центра ООО "Новое дело"

Проект разработан в рамках производственной практики студентом группы ИСиП-31.
**Роль:** Web-разработчик / UI-дизайнер: Данил М.

## 📋 Описание проекта
Веб-приложение предназначено для автоматизации процесса приема заявок на ремонт бытовой электроники и предоставления клиентам возможности самостоятельного отслеживания статуса заказа (Tracking).

## 📂 Структура проекта
* **Аналитика:** Матричное моделирование бизнес-процессов (AS-IS и TO-BE).
* **Проектирование:** Информационная архитектура (Sitemap) и Wireframes ключевых экранов.
* **Инструменты:** Figma, Markdown.
